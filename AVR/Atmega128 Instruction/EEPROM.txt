EEAR : EEPROM의 주소값 지정
EEDR : EEPROM에 쓸 데이터 저장
EEMWE : EEWE를 사용할지 결정( 1로 Set했을때 4클락 이내에 EEWE에 1을 Set한다면 지정된 주소로 뎅터를 넣는다 )
        1로 설정되었다면 EEWE가 1이든 0이든 무반응이고 4클락 후에 하드웨어가 자동으로 0으로 초기화
EEWE : EEMWE가 1일 떄 Set된다면 EEAR의 주소값으로 EEDR의 데이터를 저장(2클락 이후 자동으로 0으로 초기화)
EERE : EERE가 1로 Set된다면 데이터를 읽어오고 나서 4클락동안 정지상태가 도니다.

EEPROM에 데이터 쓰기
1. EEWE가 0인지 확인
2. SPMCSR레지스터의 SPMEN비트가 0인지 확인
3. EEAR레지스터에 데이터를 저장할 주소 지정
4. EEDR레지스터에 저장할 데이터 넣기
5. EEMWE를 1로 설정(EEWE는 0을 유지)
6. EEMWE가 1로 설정되고 나서 4클락 이내에 EEWE를 1로 설정하면 데이터가 저장된다.
 ★ Step5 이전에 전역 인터럽트를 해제하고 모두다 끝난 이후에 전역 인터럽트를 되돌리는 것이 바람직하다

EEPROM에서 데이터 읽기
1. EEWE가 0인지 확인
2. EEAR레지스터에 데이터를 불러올 주소 지정
3. EERE레지스터를 1로 지정
4. EEDR레지스터에 저장된 데이터가 있으므로 이것을 활용
